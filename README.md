# PowerLoadTester

This is a **powerful async load testing script** using Python `aiohttp` and `asyncio`.  

## Features
- Asynchronous HTTP GET requests
- Concurrency limit
- Delay between requests
- Summary report of successful and failed requests
- Custom headers support
